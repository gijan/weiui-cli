# weiui-cli

> weiui-cli 是 配合 weiui 框架使用的命令行工具

# 命令行使用

## 安装

```bash
npm i weiui-cli -g
```

## 更新

```bash
npm update weiui-cli -g
```

## 使用


#### 创建应用

```bash
weiui create [projectName]
```

- projectName: 工程名称（选题，默认：weiui_demo）


#### 查询版本

```bash
weiui list              // 显示可用的版本
```


#### 版本及帮助

```bash
weiui -v    // 查看当前toolkit版本

weiui -h    // 命令帮助信息
```
